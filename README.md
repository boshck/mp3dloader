Бот умеет искать музыку в ВК (нужны токены), качать музыку из YT и YT Music

На сервере:
cd && mkdir MusicBot && cd MusicBot
git clone git@github.com:boshck/TG-mp3dloader.git .
cp .env.example .env
nano .env

Вставь свои значения, сохрани и подними проект: 
docker compose up -d --build

Если что-то не работает - пиши, постараюсь помочь!
